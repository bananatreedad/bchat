# bchat
A nice little CLI chat tool.


##installation
1. Clone the directory to your computer
```bash
git clone https://github.com/bananatreedad/bchat
```

2. change into the bchat directory
```bash
cd bchat
```
From here you can run the server and/or client.

##run server
```bash
./gradlew runServer
```

##run client
```bash
./gradlew runServer
```
